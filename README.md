## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Install additional dependencies if required

   ```bash
   npx expo install expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants
   ```

3. Start the app

   ```bash
   npx expo start
   ```

The app directory has screens and navigation layouts and the src directory holds the logic, data, and state
